Pelatihan: Belajar HTML, CSS, dan JavaScript

Pelatihan ini dirancang untuk pemula yang ingin memahami dasar-dasar pengembangan web dengan HTML, CSS, dan JavaScript. Dalam pelatihan ini, Anda akan belajar membangun halaman web interaktif dari awal hingga selesai.

Apa yang Akan Anda Pelajari

  • Pengenalan HTML dan Struktur Dasar Halaman Web
  • Membuat Layout dengan CSS (Cascading Style Sheets)
  • Interaktivitas dengan JavaScript (DOM Manipulation, Event Handling)
  • Responsive Design dan Media Queries
  • Proyek Akhir: Membangun Website Portofolio
