Pelatihan: Mobile App Development

Pelatihan ini dirancang untuk pemula yang ingin belajar tentang pengembangan aplikasi mobile. Anda akan belajar cara membangun aplikasi untuk platform Android dan iOS menggunakan alat dan teknik terbaru.

Apa yang Akan Anda Pelajari

  • Pengenalan Pengembangan Aplikasi Mobile
  • Dasar-dasar Java dan Kotlin untuk Android
  • Swift dan Objective-C untuk iOS
  • Penggunaan Framework seperti Flutter dan React Native
  • Pengujian Aplikasi dan Debugging
  • Proyek Akhir: Membangun Aplikasi Mobile Sederhana
